## Install dependencies manually
```
brew install raylib
echo 'export C_INCLUDE_PATH="$(brew --prefix)/include:$C_INCLUDE_PATH"' >> ~/.zshrc
echo 'export LIBRARY_PATH="$(brew --prefix)/lib:$LIBRARY_PATH"' >> ~/.zshrc
```
```
Restart Shell
```
